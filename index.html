✅ Konfigurasi EDC Connection (Domain)

MODE DOMAIN (Recommended)

Gunakan subdomain *.pcsindonesia.com dengan sertifikat Sectigo wildcard. Setiap toko menggunakan subdomain unik yang diarahkan ke IP EDC via hosts file.

Contoh Setup:
  • Toko 001: edc-001.pcsindonesia.com → 192.168.1.10
  • Toko 002: edc-002.pcsindonesia.com → 192.168.1.20
  • Semua subdomain trusted oleh browser (Sectigo SSL)

🌐 Setup DNS Hosts (Windows)

💡 Kenapa perlu setup Hosts?

Domain edc-XXX.pcsindonesia.com perlu diarahkan ke IP EDC lokal toko Anda. Karena ini network internal, kita menggunakan file hosts Windows.

1

Cek IP EDC

Buka aplikasi ECR Link di EDC, catat IP address yang ditampilkan.

Contoh IP: 192.168.1.100 atau 10.0.0.50
2

Buka File Hosts

Tekan Windows + R, ketik:

notepad C:\Windows\System32\drivers\etc\hosts
Jika Notepad tidak bisa save, buka Notepad sebagai Administrator
3

Tambahkan Entry

Tambahkan baris berikut di bagian bawah file hosts:

# EDC Config
192.168.1.100 edc-001.pcsindonesia.com
4

Save & Test

  1. Save file hosts (Ctrl + S)
  2. Buka Command Prompt sebagai Administrator
  3. Ketik: ipconfig /flushdns
  4. Restart Google Chrome
  5. Test ping: ping edc-001.pcsindonesia.com
Tips
  • Pastikan Notepad dijalankan sebagai Administrator
  • Jika ipconfig /flushdns tidak work, restart komputer
  • Format: IP_ADDRESS DOMAIN (pisah dengan spasi/tab)

🔒 Setup Sertifikat SSL (WSS)

1

Download Sertifikat

Download file sertifikat CA untuk di-install ke komputer Anda.

File: pcsindonesia_ca.crt
2

Install Sertifikat di Windows

  1. Buka file sertifikat yang sudah di-download
  2. Klik "Install Certificate..."
  3. Pilih "Local Machine" → Next
  4. Pilih "Place all certificates in the following store"
  5. Klik Browse → Pilih "Trusted Root Certification Authorities"
  6. Next → Finish
  7. Restart Google Chrome

3

Verifikasi Koneksi

  • Jika masih error, coba restart komputer setelah install sertifikat
  • Pastikan IP EDC sudah benar di menu Pengaturan
