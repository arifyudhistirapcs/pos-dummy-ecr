🌐 Setup DNS Hosts

💡 Cara Kerja

Domain [subdomain].pcsindonesia.com perlu diarahkan ke IP EDC lokal toko Anda. Gunakan script otomatis di bawah ini untuk setup.

1

Cek IP EDC

Buka aplikasi ECR Link di EDC, catat IP address yang ditampilkan.

PENTING: Pastikan IP EDC adalah STATIC IP (bukan DHCP). Hubungi IT support jika perlu setting static IP.
Contoh IP: 192.168.1.100
2

Download & Jalankan Script (Windows)

Download script setup untuk Windows:

Download Script Windows (.bat)
  1. Download file setup-hosts-windows.bat
  2. Klik kanan file → "Run as administrator"
  3. Masukkan IP EDC saat diminta
  4. Ikuti instruksi di layar
  5. Script akan otomatis:
    • Backup file hosts
    • Tambahkan entry IP_EDC ecrlink.pcsindonesia.com
    • Flush DNS cache
    • Test koneksi
3

Download & Jalankan Script (Mac/Linux)

Download script setup untuk Mac/Linux:

Download Script Mac/Linux (.sh)
  1. Download file setup-hosts-macos-linux.sh
  2. Buka Terminal
  3. Jalankan: sudo bash setup-hosts-macos-linux.sh
  4. Masukkan IP EDC saat diminta
  5. Ikuti instruksi di layar
4

Test Koneksi

Setelah script selesai, test koneksi:

  1. Restart Google Chrome
  2. Buka Command Prompt / Terminal
  3. Ketik: ping ecrlink.pcsindonesia.com
  4. Jika reply dari IP EDC, berarti sukses!
Tips
  • Pastikan menjalankan script sebagai Administrator / sudo
  • Script akan membuat backup hosts file otomatis
  • Jika ada masalah, backup file ada di lokasi yang sama dengan hosts file

🔒 Setup Sertifikat SSL (WSS)

1

Download Sertifikat

Download file sertifikat CA untuk di-install ke komputer Anda.

File: pcsindonesia_ca.crt
2

Install Sertifikat di Windows

  1. Buka file sertifikat yang sudah di-download
  2. Klik "Install Certificate..."
  3. Pilih "Local Machine" → Next
  4. Pilih "Place all certificates in the following store"
  5. Klik Browse → Pilih "Trusted Root Certification Authorities"
  6. Next → Finish
  7. Restart Google Chrome
Ilustrasi langkah install
3

Verifikasi Koneksi

Setelah install sertifikat, test koneksi ke EDC:

Butuh Bantuan?
  • Pastikan Anda menggunakan browser Google Chrome
  • Jika masih error, coba restart komputer setelah install sertifikat
  • Pastikan IP EDC sudah benar di menu Pengaturan
